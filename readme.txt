* main program: India_house_price_predictions.ipynb
* utils: helper .py files, which defines customized transformer. it is only required for remote usage
* input: input data files
* ckpt: trained models
 ** ./ckpt/final_pipeline: saved the pipeline for final model
 ** ./ckpt/model_selection: saved the pipelines at the model selection step
*output: the submission.csv the prediction result over the test.csv

Major python packages have been defined in requirements.txt
pip install -r requirements.txt

The India_house_price_predictions.zip is a html demo for the India_house_price_predictions.ipynb
It allows you review the notebook offline with table of contents
